Использование с графическим интерфейсом

Начиная с версии v0.3-alpha Embark предоставляет режим пользовательского интерфейса. Если исполняемый файл Embark запущен без аргументов (простой двойной щелчок по exe-файлу), он ищет все игровые книги в текущем каталоге и предоставляет диалоговое окно, в котором вы можете выбрать нужный плейбук. Кодировка плейбука по умолчанию — UTF-8.

Сначала появится окно выбора плейбука:

Он просканирует текущий каталог и найдет все файлы, похожие на плейбук, и предложит их для запуска. Когда вы нажмете на опцию, плейбук запустится и появится окно исполнителя:

Здесь вы можете отслеживать все действия, происходящие в процессе выполнения плейбука.

Значки имеют следующее значение:

  • ✅ - задача выполнена

  • ⌛ - задача выполняется

  • ❔ - задача находится в ожидании

  • ❌ - задача не выполнена

  • ⏩ - задача пропущена
