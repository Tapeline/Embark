Использование с графическим интерфейсом

Начиная с v0.3-alpha Embark поддерживает UI режим

Если Embark запущен без аргументов командной строки (в т.ч. и по обычному двойному клику), то производится поиск всех плейбуков в текущей директории, и в открывшемся диалоге предлагается выбрать один для запуска. Кодировка в таком случае стандартная - UTF-8
