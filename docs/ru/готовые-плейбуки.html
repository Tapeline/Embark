Готовые плейбуки

Если у вас есть архив и/или папка с embark.exe, playbook.yml и множеством других файлов, то это, вероятно, плейбук, который был собран для того, чтобы вы могли быстро его развернуть.

Запуск плейбука

Чтобы запустить плейбук, просто дважды щелкните файл embark.exe. Он запросит административные разрешения, вы должны их предоставить, потому что плейбуки предназначены для изменения ваших приложений и системы, поэтому эти разрешения необходимы.

По двойному щелчку на embark.exe откроется экран:

Нажмите на нужную кнопку плейбука. Появится главный экран:

Справа можно видеть, как происходит выполнение плейбука:

  • ✅ - задача выполнена

  • ⌛ - задача выполняется

  • ❔ - задача ожидает выполнения

  • ❌ - задача не выполнена

  • ⏩ - задача пропущена
