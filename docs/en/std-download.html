std.download

Schema

url: str # source dst: str # destination overwrite: bool = False # overwrite existing file timeout_s: int = 10 # network request timeout in seconds

Action

Execution criteria

  • Task is executed if such destination file does not exist or overwrite flag set to true.

Execution requirements

  • No requirements

Target action

  • Download file from url to dst

Examples

name: Download VLC std.download: url: https://get.videolan.org/vlc/3.0.21/win64/vlc-3.0.21-win64.exe dst: vlc.exe
